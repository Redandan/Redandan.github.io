# Agora Market Windows Application Downloads

This directory contains Windows application releases for Agora Market.

## Download Latest Version

Download the latest version from the windows/ directory in this repository.

## Installation Instructions

### Method 1: Using Self-Extracting Archive (Recommended)
1. Download the .exe self-extracting archive for the version you want
2. Double-click the .exe file
3. Follow the installation prompts - it will automatically extract and install the application
4. A desktop shortcut will be created automatically

### Method 2: Using Installer Script
1. Download the .bat installer script and corresponding ZIP file for the version you want
2. Place both files in the same directory
3. Double-click the .bat file to automatically install and create a desktop shortcut

### Method 3: Manual Installation
1. Download the ZIP file
2. Extract to any directory (e.g., C:\AgoraMarket)
3. Run gora-market.exe

## Notes

- Windows may show a "Windows protected your PC" warning, click "More info" ??"Run anyway"
- First run may require administrator privileges
- It's recommended to add the application to the firewall exception list
- The self-extracting archive (.exe) requires no additional software to extract
